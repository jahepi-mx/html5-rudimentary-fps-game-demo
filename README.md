## Rudimentary FPS in JavaScript

This demo is meant to show how to implement an FPS camera and control scheme for a 3D game in JavaScript.

## Control Scheme

- w – Move forward
- s – Move backward
- a – Move left
- d – Move right
- mouse – Look around
- mouse click – Shoot
